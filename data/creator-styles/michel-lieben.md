# Michel Lieben — Creator Style Guide

> 45 posts analyzed. Agency CEO. GTM stack breakdowns. Proof-heavy.

---

## Who He Is

Founder/CEO of ColdIQ ($7M ARR). Posts about Claude Code, outbound infrastructure, GTM stacks, and agency work. Similar to Alex Vacca but more company-announcement driven. His best posts are teardowns of how other companies (lemlist, ColdIQ) actually work.

---

## Hook Patterns (real examples from his top posts)

**Pattern 1 — Question + immediate answer (his top format)**
```
Can you run your entire GTM with Claude Code?

We're answering that tomorrow.
```
Rhetorical question — but he answers it himself immediately. Doesn't leave it hanging.

**Pattern 2 — What happened yesterday (social proof)**
```
Yesterday, > 1400 people watched Kenny Damian build an entire outbound campaign using Claude code.

He wrote a few prompts in the terminal, fed it a list of companies, and then...
```
Real event + real number + what happened. The "..." creates the read-more.

**Pattern 3 — Time claim + promise**
```
Building cold email campaigns used to take hours.

Claude Code builds them in minutes:
```
Before/after with a colon that opens the breakdown. Not AI-symmetry — it's a real product claim.

**Pattern 4 — Company milestone teardown**
```
What lemlist's $42,000,000 ARR funnel looks like:

(It generated +$31M in new ARR in 2025)
```
The number is the hook. Parenthetical adds credibility.

**Pattern 5 — Operational reveal**
```
We manage > $300k/mo in ad spend via Claude Code.

These Skills 2x'd our output without sacrificing quality:
```
Specific number + what it did. Colon opens the list.

## Body Structure

**His most used format — numbered emoji sections:**
```
1️⃣ Lead Generation
[paragraph + sub-bullets with what software they use]

2️⃣ Cold Outreach (Outbound)
[paragraph + sub-bullets]
```

**→ Arrow lists for outputs/capabilities:**
```
→ ICP scoring & signal-based lead prioritization on autopilot
→ cold email sequences built from real performance data
→ AI reply handling in under 2 minutes
```

**Numbered step-by-step (for process breakdowns):**
```
1️⃣ Build your company list
[how to do it, what tools]

2️⃣ Score and tier accounts
[what it does]
```

**Closing pattern:**
Usually a link + a PS question:
```
Here is the recording and the full playbook bundle — [link]

P.S: What's the craziest buildout you've done using Claude Code?
```

---

## Voice Markers

- `> 1400 people`, `> $300k/mo` — uses `>` for approximate big numbers
- Parenthetical notes for added proof: "(The kind of campaigns that provide > 4X ROAS on > $1M spent)"
- Colons as bridges into lists — no filler transition sentences
- "He wrote a few prompts in the terminal" — plain English, no hype
- Personal detail for credibility: "Ivan Falco, our Head of Growth, built these based on 300+ hours running ad campaigns"
- Almost never talks about himself personally — it's always about what the team did

---

## CTA Style

Two patterns:
1. Link at the end: `Here's the recording — [link]`
2. PS question: `P.S: What's the craziest buildout you've done?`

Never asks for a comment keyword — that's more Alex Vacca's style.

---

## What He Never Does

- No emotional personal stories
- No em dashes
- No "it's not X, it's Y" reveals
- Doesn't hype tools without numbers attached
- Never explains what a tool is if the name is self-explanatory

---

## When to Use This Style

- Agency proof posts (what ColdIQ built, what clients got)
- Stack breakdowns (here are the 19 APIs / all the tools lemlist uses)
- Process teardowns (step-by-step how Claude Code runs a campaign)
- Webinar/event recap posts
