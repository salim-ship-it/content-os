# Sacha Martinot — Creator Style Guide

> 8 viral posts analyzed. Claude Code agent builds. Comment-trigger giveaways.

---

## Who He Is

Builder and AI automation operator. Posts about Claude Code agents, n8n workflows, and tools he's built. His format is consistent: "I built X, it does Y, comment Z to get it." Heavy comment engagement because every post includes a giveaway of the actual system. Very concrete — no theory.

---

## Hook Patterns (real examples from his top posts)

**Pattern 1 — "I built X. Y result."**
```
I built a Claude Code agent that books meetings automatically.

It's generated 3 demos this week already.
```
Built + result. Two sentences. The second one proves the first isn't theory.

**Pattern 2 — "I automated X so you don't have to"**
```
I automated my entire LinkedIn outreach with Claude Code.

Took me 4 hours to build. Runs 24/7 now.
```
Time to build + what it does now. Makes it feel achievable.

**Pattern 3 — Tool announcement**
```
Just shipped: a free n8n workflow for LinkedIn lead gen.

It does [specific thing].
```
"Just shipped" as an opener. Then what it actually does.

---

## Body Structure

**Build reveal post:**
```
[Hook: "I built X. Y result."]

Here's exactly how it works:

1️⃣ [Step with what it does]
2️⃣ [Step with what it does]
3️⃣ [Step with what it does]
4️⃣ [Step with what it does]

[What it took / what it replaced]

Comment "[WORD]" and I'll DM you the full [thing].
```

**The numbered emoji steps are his signature** — 1️⃣ 2️⃣ 3️⃣ 4️⃣ — not plain numbers, not bullets.

---

## Voice Markers

- "I built" as opener — never "We built" or "Our team built"
- Numbered emoji steps (1️⃣ 2️⃣ 3️⃣) for how-it-works sections
- "Took me X [time]" — gives it a human build cost
- "Runs 24/7 now" — the payoff phrase after the build cost
- No fluff before the build reveal
- Concrete outputs: "3 demos", "47 leads", not "many meetings"

---

## CTA Style

Always a comment trigger, always for the actual system:

```
Comment "AGENT" and I'll DM you the full Claude Code setup.
```

Sometimes adds:
```
No pitch. Just the workflow.
```

The "No pitch. Just X." line is a trust-builder he uses often.

---

## What He Never Does

- No motivation or inspiration angle
- Never shares failures or vulnerable moments
- No em dashes
- Doesn't write philosophical observations about AI
- Never writes general advice — always about a specific thing he built

---

## When to Use This Style

- Claude Code agent reveal posts
- n8n / Make.com automation builds
- "I automated X" posts with a free giveaway
- Tool launches or system reveals
